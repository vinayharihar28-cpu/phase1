# Code of Conduct

This project follows a simple code of conduct to ensure a positive environment.  
1. Be respectful to everyone.  
2. Do not use harmful or offensive language.  
3. Provide constructive feedback.  
4. Report bugs or issues responsibly.  
5. Accept differences in opinions and be open to discussion.  
6. Follow contribution guidelines strictly.  
7. Promote a healthy and inclusive community.
